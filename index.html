📦 مكونات المشروع الرئيسية

🧠 cpu_memory.py - قياسات المعالج والذاكرة

  • قياس سرعة تخصيص الذاكرة
  • قياس سرعة الوصول للذاكرة (متسلسل وعشوائي)
  • اختبار أداء CPU في العمليات الحسابية
  • مراقبة استخدام الذاكرة على المدى الطويل

⚙️ process_management.py - إدارة العمليات

  • قياس سرعة إنشاء العمليات
  • قياس تبديل السياق (Context Switching)
  • اختبار إنشاء الخيوط (Threading)
  • تقييم كفاءة إدارة العمليات

📁 filesystem.py - نظام الملفات

  • قياس سرعة إنشاء الملفات
  • قياس سرعة قراءة الملفات
  • قياس سرعة كتابة الملفات والإنتاجية
  • اختبار عمليات المجلدات والديريكتوريات

🔒 security.py - الأمان والتشفير

  • قياس سرعة عمليات Hashing (MD5, SHA256, SHA512)
  • محاكاة عمليات التشفير
  • فحص ميزات الأمان (ASLR)
  • التحقق من أذونات الملفات

⚙️ config.py - الإعدادات

  • معلومات النظام التلقائية
  • إعدادات Benchmark قابلة للتخصيص
  • معايير الأداء
  • إدارة المجلدات والملفات

📊 data_analyzer.py - تحليل البيانات

  • توليد تقارير HTML شاملة
  • تنسيق النتائج بصيغة JSON
  • عرض معلومات النظام
  • تحليل النتائج ومقارنتها

📝 logger.py - تسجيل السجلات

  • تسجيل شامل للعمليات
  • تتبع الأخطاء والتحذيرات
  • حفظ السجلات للمراجعة

🚀 main.py - البرنامج الرئيسي

  • تنسيق تشغيل جميع الاختبارات
  • إدارة تدفق العمل
  • حفظ النتائج وتوليد التقارير
  • عرض رسائل التقدم والحالة
